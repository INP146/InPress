# Style extensions

InPress includes a small set of content and navigation styles that are controlled through CSS custom properties in the consuming site's theme stylesheet.

## Text markers

Use the underline marker for short emphasis and the brush marker for highlighted text:

<p><mark>Underline marker</mark></p>
<p><mark class="highlight">Brush-style marker highlight</mark></p>

```html
<mark>Underline marker</mark>
<mark class="highlight">Brush-style marker highlight</mark>
```

Both markers use the VitePress brand tokens, including the palette generated by [`color`](/config/color). Override their colors with `--inpress-marker-color` and `--inpress-marker-highlight-color`. Adjust their geometry with `--inpress-marker-thickness`, `--inpress-marker-offset`, and `--inpress-marker-highlight-radius`.

## Sidebar hierarchy

First-level page links are indented by `16px` below their group heading. Override the spacing in the consuming site's stylesheet:

```css
:root {
  --inpress-sidebar-indent: 20px;
}
```
