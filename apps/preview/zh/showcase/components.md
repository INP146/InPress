# 链接功能

## 平台链接

平台链接图标默认开启。可通过 `themeConfig` 为站点关闭，或指定只启用部分平台：

```ts
// .vitepress/config.ts
import { defineConfig } from "vitepress";

export default defineConfig({
  themeConfig: {
    linkIcons: false,
    // 或：linkIcons: ['github', 'youtube']
  },
});
```
