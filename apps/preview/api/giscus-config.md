# GiscusConfig

- Import: `@inp146/inpress`
- Kind: TypeScript interface

Describes the Giscus widget configured by [`giscus`](/config/giscus).

| Field | Type | Required | Default |
| --- | --- | --- | --- |
| `repo` | `string` | yes | - |
| `repoId` | `string` | yes | - |
| `category` | `string` | yes | - |
| `categoryId` | `string` | yes | - |
| `mapping` | `GiscusMapping` | no | `'pathname'` |
| `term` | `string` | no | - |
| `strict` | `boolean` | no | `false` |
| `reactionsEnabled` | `boolean` | no | `true` |
| `emitMetadata` | `boolean` | no | `false` |
| `inputPosition` | `'top' | 'bottom'` | no | `'bottom'` |
| `theme` | `string | GiscusTheme` | no | follows appearance |
| `lang` | `string` | no | current page language |
| `loading` | `'lazy'` | no | browser default |

Use the repository, category, and behavior values generated by [giscus.app](https://giscus.app). The `term` field is used by mapping modes that require an explicit discussion term.
