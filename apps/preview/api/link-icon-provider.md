# LinkIconProvider

- Import: `@inp146/inpress`
- Kind: TypeScript union

Represents one provider supported by [`linkIcons`](/config/link-icons).

```ts
type LinkIconProvider =
  | 'github'
  | 'gitlab'
  | 'npm'
  | 'discord'
  | 'telegram'
  | 'linkedin'
  | 'reddit'
  | 'twitch'
  | 'tiktok'
  | 'weibo'
  | 'xiaohongshu'
  | 'zhihu'
  | 'juejin'
  | 'x'
  | 'instagram'
  | 'threads'
  | 'youtube'
  | 'bilibili'
```

The union is derived from [`linkIconProviders`](/api/link-icon-providers), keeping the runtime list and TypeScript type synchronized.
